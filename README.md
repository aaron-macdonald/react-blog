# Boilerplate: Webpack - Babel - React
Introductory template using webpack, babel and react - April 2019

## Getting Started

* Fork this repo
* Rename your repo accordingly.

````
git clone https://github.com/[your-org]/[your-app].git
cd [your-app]
npm i
````

This is the structure of this boilerplate:

```sh
.
├── dist
│   └── index.js
│   ├── main.js
├── src
│   ├── js
│   │   └── components
│   │       └── container
│   │       │    └── FormContainer.jsx
│   │       └── presentational
│   │            └── input.jsx
│   ├── index.html
│   └── index.js
├── .babelrc
├── .gitignore
├── LICENSE
├── package-lock.json
├── package.json
├── README.md
├── SETUP.md
└── webpack.config.js
```
